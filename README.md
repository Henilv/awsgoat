# awsgoat/pacu- aws infra security testing

AWSGoat is a vulnerable-by-design AWS infrastructure exposing OWASP Top 10 (2021) risks and misconfigurations across IAM, S3, API Gateway, Lambda, EC2, and ECS. Using Terraform and GitHub Actions, it enables cloud pentesting, IaC auditing, secure coding, detection, and mitigation in realistic modular environments.It features multiple escalation paths and is focused on a black-box approach.

The project has pacu.yaml which is aws offensive framework for testing, it has response/requests with redacted payload/info.
best-practices.config is security practices when achieved provided minimal security guarantees against adversary.
Security takeaway gives a literature walkthrough.

More can be found out at: https://github.com/ine-labs/AWSGoat?tab=readme-ov-file
